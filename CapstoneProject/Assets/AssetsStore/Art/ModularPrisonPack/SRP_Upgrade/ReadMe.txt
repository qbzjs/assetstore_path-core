

Universal Render Pipeline upgrade
---------------------------------
Supported version: 7.1.8 and up only!

1) Create a URP project, use Package Manager to add Lightweight RP component to Unity (important!)
2) Use project settings menu - set Lightmap Encoding to High Quality!
2) Import Modular Prison Pack, and simply run URP_Upgrade package, and done!
