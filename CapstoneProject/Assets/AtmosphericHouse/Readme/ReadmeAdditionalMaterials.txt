Atmospheric House - Additional Materials:

HDRP & URP instructions:
1. Import Atmospheric House (Modular) to your desired render pipeline project.
2. From HDRP or URP folder, import AtmosphericHouseHDRP/URP.unitypackage file to your project.
3. Import Additional Materials to the same project.
4. From HDRP or URP folder, import AdditionalMaterialsHDRP/URP.unitypackage file to your project.
5. Use Linear Color Space for shaders to work properly.

Additional textures in folder Textures_additional
Additional materials in folder Materials -> Materials_additional
